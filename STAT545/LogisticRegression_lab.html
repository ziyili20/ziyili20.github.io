Get our data

We download our data from the UCLA data web portal. This dataset has a binary response (outcome, dependent) variable called admit. There are three predictor variables: gre, gpa and rank. We will treat the variables gre and gpa as continuous. The variable rank takes on the values 1 through 4. Institutions with a rank of 1 have the highest prestige, while those with a rank of 4 have the lowest. We can get basic descriptives for the entire data set by using summary. To get the standard deviations, we use sapply to apply the sd function to each variable in the dataset.

add confidence interval to the fitted line

library(ggplot2)
fitdat <- data.frame(gpa = rep(seq(2.3, 4, by = 0.1), 4),
                     gre = rep(600, 72),
                     rank = rep(1:4, each = 18))

newfit <- predict(mylogit, newdata = fitdat, type = "link",
    se = TRUE)
newdata <- within(newfit, {
    PredictedProb <- plogis(fit)
    LL <- plogis(fit - (1.96 * se.fit))
    UL <- plogis(fit + (1.96 * se.fit))
})
newdata2 <- cbind(fitdat, newdata)
